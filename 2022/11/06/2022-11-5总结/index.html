军训

Description

起点 (0,0)(0,0)(0,0) ，每次只能走恰好曼哈顿距离为 kkk 的点，构造一种到达 (x,y)(x,y)(x,y) 且步数最小的方案，无解输出 -1。

Input

一行三个整数 k,x,yk, x, yk,x,y 。

Output

若无解，一行一个整数 -1。

否则第一行一个整数 sss, 代表最小操作次数。

接下来 sss 行，第 iii 行两个整数 xi,yix_i, y_ixi​,yi​, 表示第 iii 次操作后的坐标，若多组解任意输出一组。

Hint

1≤k≤109,−105≤x,y≤105,(x,y)≠(0,0)1 \le k \le 10^9, -10^5 \le x,y \le 10^5, (x,y) \not = (0,0)1≤k≤109,−105≤x,y≤105,(x,y)=(0,0)

Solution

因为 x,yx,yx,y 可正可负，不如先将 x,yx,yx,y 取绝对值，输出时判符号。

如果 kkk 是奇数，那么所有坐标都可达(存在一种方案从 (x,y)(x,y)(x,y) 走到 (x+1,y)(x+1,y)(x+1,y) )。

如果 kkk 是偶数，那么 x+y≡1(mod2)x+y \equiv 1 \pmod 2x+y≡1(mod2) 的所有坐标都不可达，其他坐标都可达(存在一种方案从 (x,y)(x,y)(x,y) 走到 (x+1,y+1)(x+1,y+1)(x+1,y+1) )

如果 x+y>2kx+y > 2kx+y>2k ，那么直接向 (x,y)(x,y)(x,y) 的方向走即可。

如果 x+y≤2kx+y \le 2kx+y≤2k ，此时一定有 2k−x−y mod 2=02k-x-y \bmod 2 = 02k−x−ymod2=0，设 x+y+2t=2kx+y+2t = 2kx+y+2t=2k，则 t=k−⌊x+y2⌋t = k-\lfloor \frac{x+y}{2} \rfloort=k−⌊2x+y​⌋，然后我们分两步走，每步在其中一个反方向走 ttt，这样两次走的反方向就是 2t2t2t ，走过的距离就是 x+y+2t=kx+y+2t=kx+y+2t=k ，正好能从 (x,y)(x,y)(x,y) 走到 (0,0)(0,0)(0,0)

命题规范

Description

有 nnn 种需要满足的命题规范，mmm 个出题人，提供 ai,ja_{i,j}ai,j​ 个大样例可以让第 jjj 个出题人满足第 iii 个规范，在找到第 iii 个出题人满足规范之前，需要先提供 bib_ibi​ 个大样例。

求满足所有命题规范最少提供多少大样例。

Input

第一行两个整数 n,mn,mn,m

接下来 nnn 行，第 iii 行 mmm 个整数 ai,1,ai,2,⋯ ,ai,ma_{i,1},a_{i,2},\cdots,a_{i,m}ai,1​,ai,2​,⋯,ai,m​。

接下来一行 mmm 个整数 b1,b2,⋯ ,bmb_1, b_2, \cdots, b_mb1​,b2​,⋯,bm​

Output

一行一个整数表示答案

Hint

1≤n≤105,1≤m≤25,1≤ai,j,bi≤1091 \le n \le 10^5, 1 \le m \le 25, 1 \le a_{i,j},b_i \le 10^91≤n≤105,1≤m≤25,1≤ai,j​,bi​≤109

Solution

直接暴力是 O(nm2m)O(nm2^m)O(nm2m) 的，首先 2m2^m2m 没办法省掉，那么需要避免每次枚举 nmnmnm 来计算答案。

如果还是保留暴力的取 min⁡\minmin ，是没办法降低时间复杂度的，因为取 min⁡\minmin 无法继承贡献。

考虑出题人选定时每道题的贡献: 最小的出现在出题人集合里的数。

一个小 trick: 差分数组的前缀和是原数，这样的话我们可以把取 min⁡\minmin 变成求和操作，方便继承贡献。

对于一道题 iii ，设 ai,p1≤ai,p2≤⋯≤ai,pma_{i,p_1} \le a_{i,p_2} \le \cdots \le a_{i,p_m}ai,p1​​≤ai,p2​​≤⋯≤ai,pm​​，设选择的出题人集合为 SSS ，对于 1≤j≤m1 \le j \le m1≤j≤m，若 {p1,p2,⋯ ,pj−1}⊆{1,2,⋯ ,m}∖S\{p_1, p_2, \cdots, p_{j-1}\} \subseteq \{1,2,\cdots,m\} \setminus S{p1​,p2​,⋯,pj−1​}⊆{1,2,⋯,m}∖S ，则答案需要加上 ai,pj−ai,pj−1a_{i,p_j}-a_{i,p_{j-1}}ai,pj​​−ai,pj−1​​，我们将这个数作为集合 {p1,p2,⋯ ,pj−1}\{p_1, p_2, \cdots, p_{j-1}\}{p1​,p2​,⋯,pj−1​} 的贡献。

出题人集合 SSS 的答案即为 {1,2,⋯ ,m}∖S\{1,2,\cdots,m\} \setminus S{1,2,⋯,m}∖S 的子集和。

时间复杂度 O(nmlog⁡m+m2m)O(nm \log m + m2^m)O(nmlogm+m2m)。

前置知识

Description

给定两个长度为 nnn 的序列 a,ba,ba,b，求一个 111 到 nnn 的排列 ccc ，最小化:

∑i=1n⌈max⁡(bi−aci,0)k⌉\sum_{i=1}^{n} \left\lceil \frac{\max(b_i-a_{c_i}, 0)}{k} \right\rceil i=1∑n​⌈kmax(bi​−aci​​,0)​⌉

Input

第一行两个整数 n,kn, kn,k。

第二行 nnn 个整数 a1,a2,⋯ ,ana_1, a_2, \cdots, a_na1​,a2​,⋯,an​。

第三行 nnn 个整数 b1,b2,⋯ ,bnb_1, b_2, \cdots, b_nb1​,b2​,⋯,bn​。

Output

第一行一个整数表示答案。

第二行 nnn 个整数 c1,c2,⋯ ,cnc_1, c_2, \cdots, c_nc1​,c2​,⋯,cn​。

若有多组解任意输出一组即可。

Hint

1≤n≤105,1≤ai,bi,k≤1091 \le n \le 10^5, 1 \le a_i,b_i,k \le 10^91≤n≤105,1≤ai​,bi​,k≤109

Solution

考虑一个贪心。

首先问题可以看成每次操作把某个 bib_ibi​ 减去 kkk ，求最少次数使得存在排列 ccc 满足 bi≤acib_i \le a_{c_i}bi​≤aci​​。

对比当前 a,ba,ba,b 中最大值，如果 max⁡{a}≥max⁡{b}\max\{a\} \ge \max\{b\}max{a}≥max{b}，那么直接把这两个数配成一对即可，否则把 bbb 种的最大值减去 kkk。

用 map 和 set 加速贪心过程即可，具体就是把 ⌊bik⌋\lfloor \frac{b_i}{k} \rfloor⌊kbi​​⌋ 相同的 bib_ibi​ 一起处理，时间复杂度 nlog⁡2nn \log^2 nnlog2n。
