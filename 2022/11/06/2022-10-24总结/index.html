Medals

Description

你有 NNN 个朋友，他们会来你家玩，第 iii 个人 1∼Ai1 \sim A_i1∼Ai​ 天来玩，然后 Ai+1∼2AiA_i+1 \sim 2A_iAi​+1∼2Ai​ 天不来，然后 2Ai+1∼3Ai22A_i+1 \sim 3A_i22Ai​+1∼3Ai​2 又会来，以此类推。

每天你会选一个来玩的人，给他颁个奖，如果没人来玩，你就不颁奖。

你要给每个人都颁 KKK 个奖，问至少需要多少天。

Input

NNN KKK

A1,A2,⋯ ,AnA_1, A_2, \cdots, A_nA1​,A2​,⋯,An​

Output

答案

Hint

N≤18,K≤105,Ai≤105N \le 18, K \le 10^5, A_i \le 10^5N≤18,K≤105,Ai​≤105

Solution

天数的上界是 2NK2NK2NK，然后可以二分，用网络流 check，然而跑不动。

我们可以拆点看成二分图匹配问题，然后利用霍尔定理判断，进行一个超集和的统计。

Google Code Jam

Description

一场考试的时间为 TTT ，有 nnn 道题，每道题都可以花 t1it1_it1i​ 的时间获得 s1is1_is1i​ 的暴力分，并且可以在打完暴力后再花 t2it2_it2i​ 的时间多获得 s2is2_is2i​ 的分，但有 pip_ipi​ 的概率写挂正解，写挂可以不提交。求最大期望得分以及最大期望得分下的最小期望罚时（这里罚时定义为最后一次提交代码的时间）。

Input

第一行输入两个整数 nnn 和 TTT。
接下来 nnn 行每行五个数 s1i,s2i,t1i,t2i,pis1_i,s2_i,t1_i,t2_i,p_is1i​,s2i​,t1i​,t2i​,pi​，其中 s1i,s2i,t1i,t2is1_i,s2_i,t1_i,t2_is1i​,s2i​,t1i​,t2i​ 为整数，pip_ipi​ 为浮点数。

Output

输出一行两个实数，绝对或相对误差小于 10−910^{-9}10−9 算作正确。

Hint

1≤n≤1000,1≤T≤1560,t1i,t2i≤1560,s1i,s2i≤109,0≤pi≤11 \le n \le 1000, 1 \le T \le 1560, t1_i,t2_i \le 1560, s1_i,s2_i \le 10^9, 0 \le p_i \le 11≤n≤1000,1≤T≤1560,t1i​,t2i​≤1560,s1i​,s2i​≤109,0≤pi​≤1 且 pip_ipi​ 最多有 666 为小数。

Solution

设 fif_ifi​ 为花费时间为 iii 时最大期望得分，gig_igi​ 为最小期望罚时。

首先因为罚时的存在，所以每道题必定时先打暴力再打正解，然后因为正解打挂了可以不交，所以两道题顺序不一样的话期望罚时也不一样，我们可以列个式子排序一下就行，然后最大期望得分可以直接背包。
