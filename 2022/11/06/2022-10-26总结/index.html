Infinite Operation

Description

给出 nnn 和一个长度为 nnn 的数列 a1…na_{1 \dots n}a1…n​。

有 qqq 次修改，每次给出两个数 ppp、vvv，表示将 apa_pap​ 修改为 vvv。修改会改变 aaa，对以后的答案产生影响。

在每一次修改后，求出下面问题的答案：

你可以操作这个序列任意多次。下面的 aaa 与 xxx 均为实数。

每次选择两个 [1,n][1,n][1,n] 内的正整数 iii、jjj 满足 ai≤aja_i \le a_jai​≤aj​。然后选择一个 0≤x≤aj−ai20 \le x \le \frac{a_j-a_i}{2}0≤x≤2aj​−ai​​。

使 ai=ai+x,aj=aj−xa_i=a_i+x, a_j=a_j-xai​=ai​+x,aj​=aj​−x，同时得到 xxx 的分数。

求若干次操作后，你能得到的得分的最大值，在模 998244353998244353998244353 意义下的值。可以证明这个最大值是存在的且为有理数。

问题中进行的操作不会真的改变 aaa 的值，这些操作将在这次询问结束后被撤销。而进行的 qqq 次修改会改变 aaa 的值并保留下来，影响之后的答案。

Input

输入的第一行包含两个数 nnn、qqq，含义如上。

输入的第二行包含 nnn 个数，第 iii 个表示 aia_iai​，含义如上。

接下来 qqq 行，其中第 iii 行包含两个数 pip_ipi​、viv_ivi​，表示一次修改。

Output

输出应包含 qqq 行，第 iii 行包含一个数，表示你对第 iii 次修改后的 aaa 求出的答案在模 998244353998244353998244353 意义下的值。

Hint

2≤n≤3×105,1≤ai,vi≤109,1≤pi≤n2 \le n \le 3 \times 10^5, 1 \le a_i,v_i \le 10^9, 1 \le p_i \le n2≤n≤3×105,1≤ai​,vi​≤109,1≤pi​≤n。

Solution

设 S=∑i=1n∑j=i+1n∣ai−aj∣S = \sum\limits _{i=1}^n \sum\limits _{j=i+1}^n |a_i-a_j|S=i=1∑n​j=i+1∑n​∣ai​−aj​∣，显然得分最大值不大于 S2\frac{S}{2}2S​，因为每次操作会将 SSS 减去至少 2x2x2x，然后将得分加上 xxx，下面给出一组操作方式使得得分达到 s2\frac{s}{2}2s​。

考虑每次操作会将 ai,aja_i,a_jai​,aj​ 靠近它们的平均数，如果存在一个 aka_kak​ 满足 ai≤ak≤aja_i \le a_k \le a_jai​≤ak​≤aj​ ，那么这次操作后 SSS 的减少量会大于 2x2x2x，如果不存在这样的 aka_kak​ 那么 SSS 的减少量就等于 2x2x2x。

所以我们每次选择两个排序后相邻的数操作即可使得分达到 s2\frac{s}{2}2s​。

Random IS

Description

给一个长度为 NNN 的排列 AAA。

现在你要在序列上选一些数。

当前局面下，一个数可选当且仅当选完该数后已选的数构成的子序列单调递增。

若有可选的数，等概率选出一个，否则结束。

问期望选出多少个数。

NNN
