正方形

Description

给定一个 n∗mn*mn∗m 的棋盘，其中有些格子是障碍，有 qqq 次询问，每次询问给出 x1,y1,x2,y2,wx1,y1,x2,y2,wx1,y1,x2,y2,w，问能否将以 (x1,y1)(x1,y1)(x1,y1) 为右下角的边长为 www 的正方形通过平移使得右下角移动到 x2,y1x2,y1x2,y1，期间不穿过障碍且不能超出棋盘。

Input

第一行三个整数 n,m,qn,m,qn,m,q。

接下来 nnn 行，每行一个长度为 mmm 的 010101 字符串，其中第 jjj 个字符表示 (i,j)(i,j)(i,j) 是否有障碍，111 表示障碍。

接下来 qqq 行，每行五个整数 x1,y1,x2,y2,wx_1,y_1,x_2,y_2,wx1​,y1​,x2​,y2​,w。

Hint

n,m≤103,q≤105n,m \le 10^3, q \le 10^5n,m≤103,q≤105

Solution

先 O(nm)O(nm)O(nm) 处理出以每个格子为右下角能放下的最大正方形 sizi,jsiz_{i,j}sizi,j​，首先小的正方形能走的格子肯定比大的正方形多，所以我们将询问按 www 从大到小排序，每次只加入 sizi,j≥wsiz_{i,j} \ge wsizi,j​≥w 的格子，然后判断 (x1,y1)(x1,y1)(x1,y1) 与 (x2,y2)(x2,y2)(x2,y2) 是否联通，这个可以并查集维护，注意特判起点终点相同的情况。

Description

EEE：到根距离为偶数的所有非叶子节点的集合。

OOO：到根距离为奇数的所有非叶子节点的集合。

LLL：所有叶子节点的集合。

对于所有叶子节点 x∈Lx \in Lx∈L ，都会被分配一个二元组 (c(x),d(x))(c(x), d(x))(c(x),d(x))。

游戏开始前，Alice 对于 EEE 中的每个点 xxx，从两个儿子中选择一个作为重儿子，类似地，Bob 对于 OOO 中的每个点 xxx ，选择一个儿子作为重儿子。

当策略确定之后，从根开始，不断沿着重儿子走，直到走到某个叶子节点 xxx，此时 Alice 得分 c(x)c(x)c(x)，Bob 得分 d(x)d(x)d(x)。
